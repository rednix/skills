## 数据到文字的转换规则

### 收益率描述

| 收益率范围 | 描述词 |
|-----------|--------|
| > 5% | 大涨、飙升 |
| 1% ~ 5% | 上涨、涨幅 |
| 0% ~ 1% | 微涨、小幅上涨 |
| -1% ~ 0% | 微跌、小幅下跌 |
| -5% ~ -1% | 下跌、跌幅 |
| < -5% | 大跌、重挫 |

### 排名描述

| 排名 | 描述 |
|-----|------|
| 第1名 | 领涨、涨幅最大 |
| 前3名 | 涨幅居前 |
| 倒数第1 | 领跌、跌幅最大 |
| 后3名 | 跌幅居前 |

### 占比描述

| 占比范围 | 描述 |
|---------|------|
| > 80% | 绝大多数 |
| 60% ~ 80% | 超六成 |
| 50% ~ 60% | 超五成 |
| 40% ~ 50% | 近半数 |
| 20% ~ 40% | 约1/3 |
| < 20% | 少数 |

## 话术风格

**生成的周报话术符合分析师风格：**

1. **数据密集** - 一段话包含多个数据点
2. **对比结构** - 用"与之相对"连接正反两面
3. **具体名称** - 提到具体基金名称和公司
4. **专业术语** - PE-TTM、分位水平、净流入/流出
5. **结构清晰** - 先总后分，先涨后跌

**话术要点：**
- 用"巨大"而不是"较大"
- 用"近X%"而不是"超X%"（更保守）
- 用"在X%上下"而不是"超过X%"
- 用"强势领涨"而不是"领涨"
- 用"表现落后"而不是"领跌"

**示例：**
```
最近一周（0112-0116），普通股票/偏股混合基金周收益率中位数分别为1.21%/1.38%；首尾基金收益差异巨大，头部绩优产品周收益超10%，尾部产品周跌幅近5%。

TMT主题基金强势领涨，平均涨幅高达11.92%，医药生物和军工主题基金平均涨幅在10%左右；金融地产主题基金表现落后，平均涨幅仅为1.00%。
```
