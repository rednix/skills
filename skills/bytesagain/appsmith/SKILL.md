# Appsmith

Platform to build admin panels, internal tools, and dashboards. Integrates with 25+ databases and any API. ## Commands

- `help` - Help
- `run` - Run
- `info` - Info
- `status` - Status

## Features

- Core functionality from appsmithorg/appsmith

## Usage

Run any command: `appsmith <command> [args]`

## Examples

```bash
# Show help
appsmith help

# Run
appsmith run
```

## Configuration

Set `APPSMITH_DIR` to change data directory. Default: `~/.local/share/appsmith/`
